## Getting Started

### Install Dependency
```bash
npm install
# or
yarn 
# or
pnpm intsall
```

### Dev Server
To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
